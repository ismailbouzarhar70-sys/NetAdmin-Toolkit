Admin Tools

Commandes rapides et mémos pour l'administration IT

Diagnostic Réseau

Vérifier la configuration IP (Windows)
ipconfig /all
Tester la connectivité et latence
ping 8.8.8.8
Tracer la route vers une cible
tracert google.com

Commandes Linux

Mettre à jour le système
sudo apt update && sudo apt upgrade
Changer les permissions (Lecture/Ecriture/Exec)
chmod 755 nom_du_fichier
Vérifier l'état d'un service
systemctl status apache2
